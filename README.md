# naive-bayes-python
Classifier of programming languages.

Full data set: [data.csv](https://yadi.sk/d/m-a4LhmV4zQNHg).

Original data set is taken from [kaggle](https://www.kaggle.com/arjoonn/codechef-competitive-programming).

Requirements:
    - Python 3
